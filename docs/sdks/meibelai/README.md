## root_get

Root

### Example Usage

<!-- UsageSnippet language="python" operationID="root__get" method="get" path="/" -->
```python
from meibelai import Meibelai
import os


with Meibelai(
    api_key_header=os.getenv("MEIBELAI_API_KEY_HEADER", ""),
) as m_client:

    res = m_client.root_get()

    # Handle response
    print(res)

```
